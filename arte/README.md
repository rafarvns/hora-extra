# Arte - Hora-Extra 🎨

Esta pasta é o repositório central para todos os ativos visuais e sonoros do projeto.

---

## 📁 Estrutura de Pastas Sugerida
- 📂 `3D_Models/`: Personagens (estagiários, vilões) e objetos de cenário (cubículos, mesas).
- 📂 `Textures/`: Mapas de textura e materiais.
- 📂 `UI_UX/`: Mockups de interface, ícones, fontes e HUD.
- 📂 `Concepts/`: Arte conceitual e referências visuais de ambiente.
- 📂 `Audio/`: Efeitos sonoros de escritório, trilhas de tensão e dublagens.

---

## ⚠️ Regras para Artistas
1. **Padrão de Nomeclatura**: Use nomes descritivos em Snake_Case ou PascalCase (ex: `Estagiario_Variant_01` ou `Mesa_Escritorio`).
2. **Formatos**: Prefira `.fbx` para modelos, `.png` ou `.tga` para texturas.
3. **Escala**: Verifique se os modelos estão na escala correta antes de exportar para o Unity.
