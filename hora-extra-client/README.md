# Hora-Extra - Client (Unity) 🎮

Este é o projeto do cliente do jogo **Hora-Extra**, desenvolvido com o motor Unity.

---

## 🛠️ Especificações Técnicas
- **Versão do Unity**: `6000.3.12f1` (Unity 6)
- **Plataformas**: PC
- **Estética**: Semi-realista / Low-poly com iluminação estilizada.
- **Render Pipeline**: (Assumindo standard, mas se for URP/HDRP pode ser atualizado aqui)

---

## 🚀 Como Começar
1. Certifique-se de ter o **Unity Hub** instalado.
2. Adicione este projeto (`hora-extra-client`) ao seu Unity Hub.
3. Certifique-se de que a versão `6000.3.12f1` esteja instalada.
4. Abra o projeto e certifique-se de que a cena principal esteja em `Assets/Scenes`.

---

## 📁 Estrutura de Pastas
- `Assets/Scenes`: Níveis e salas do escritório.
- `Assets/Scripts`: Lógica principal do jogo (C#).
- `Assets/Models`: Modelos 3D dos estagiários e pesadelos corporativos.
- `Assets/Materials`: Texturas e materiais.
- `Assets/Prefabs`: Objetos prontos para uso em cena.

---
